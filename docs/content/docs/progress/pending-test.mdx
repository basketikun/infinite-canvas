# 待测试

- Token 一键登录：Canvas 未登录时显示一键登录入口，使用 Authorization Code、`state` 和 PKCE S256 跳转 Token/New API；Token 登录后自动创建或复用 `image` 分组令牌并返回 Canvas。
- 安全会话：Canvas 使用 Host-only、Secure、HttpOnly、SameSite=Lax Cookie；Session ID 在 SQLite 中只存 SHA-256 哈希，真实 Token API Key 使用 AES-256-GCM 加密保存，需验证数据库、浏览器存储、URL 和网络响应中均无明文密钥。
- 固定 AI 代理：所有模型请求统一走同源 `/api/ai`，服务端固定转发到 Token/New API 并覆盖浏览器 Authorization；需验证模型列表、文本流式响应、图片生成与编辑、音频、视频任务和视频内容下载。
- 配置迁移：配置页不再显示 Base URL、API Key、渠道或调用格式；旧浏览器配置和旧 URL 参数中的密钥会被清理，登录后自动刷新 Token 可用模型。
- 登录生命周期：需验证未登录跳转、已登录直接访问、OAuth 超时/篡改、退出登录、停用自动令牌、会话过期以及跨站请求来源校验。
- Docker 部署：会话数据库持久化到 `/app/data`，GitHub Actions 发布 `latest` 与 `sha-...` 镜像；需验证容器重启后会话仍有效。

- Codex App 插件文档：新增插件安装教程，区分 AI 自动安装、手动安装和本仓库开发调试安装；同步优化插件 README 与技能说明。
