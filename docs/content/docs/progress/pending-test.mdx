# 待测试

- Token 多分组一键登录与模型目录：登录后自动创建或复用 `Image`、`Video`、`ChatGPT` 三个分组令牌，浏览器不接触真实密钥；生图仅显示 `gpt-image-2` 和 `gemini-3.1-flash-image` 及 1K/2K/4K 价格，视频显示 `Video` 分组全部模型及按秒/按条价格，文本使用 `ChatGPT` 分组，音频默认 `gpt-4o-audio-preview`。需验证首次重新登录、默认模型、价格展示、刷新不再重复请求或触发 429，以及各能力请求使用正确分组。
- 固定 AI 代理剩余能力：模型列表和 `/v1/images/generations` 已完成生产验证；视频任务已修复长耗时任务过早超时、New API `done` 成功状态和直链结果兼容，并会自动恢复查询此前因超时标记失败的任务；仍需验证文本流式响应、图片编辑、音频，以及视频任务自动恢复和结果播放/下载。
- Sora/Veo 视频参数限制：`sora-2`、`sora-2-pro`、`veo-3-fast`、`veo-3` 已按接口文档限制可选时长、比例和分辨率，并在提交前校验提示词不超过 4000 字符、参考图最多 1 张且不超过 10MB、格式仅限 PNG/JPEG/WebP；需分别验证合法参数可提交，以及非法时长、比例、分辨率和参考图会显示明确提示。
- 配置迁移剩余场景：配置页已不再显示 Base URL、API Key、渠道或调用格式；仍需验证旧浏览器配置和旧 URL 参数中的密钥清理。
- 登录生命周期剩余场景：未登录跳转、已登录访问、退出登录、跨站来源校验和容器重启持久性已验证；仍需验证 OAuth 超时/篡改、停用自动令牌和会话自然过期。

- Codex App 插件文档：新增插件安装教程，区分 AI 自动安装、手动安装和本仓库开发调试安装；同步优化插件 README 与技能说明。
