# 待测试

- 通过局域网 IP 访问前台提示词、素材库、我的素材和后台相关页面时，复制文本应能正常写入剪贴板。
- 管理后台「用户管理」CRUD：新增/编辑/删除用户，角色和生图额度修改后立即生效；调整额度后流水里出现 `admin_adjust` 记录。
- 管理后台「模型配置」CRUD：新增、编辑、删除、启用切换、连通测试均正常；API Key 编辑留空保持原值，删除已启用配置应被拒绝。
- 普通用户额度扣减：生图成功后顶部「积分」实时减少，余额为 0 时再次调用提示「额度不足，请联系管理员」；个人中心流水出现 `consume` 记录。
- 管理员账号生图：顶部显示「∞」，不消耗额度，不写流水。
- 文本问答限流：1 分钟内连续触发 6 次，第 6 次应被拒绝并提示等待秒数；管理员不受限。
- 注册流程：开放注册后，新账号默认 4 点额度；个人中心首条流水为 `signup_bonus +4`；密码不一致、用户名重复时提示清晰。
- 前台「管理后台」入口：仅管理员账号在用户菜单中可见；普通用户直接访问 `/admin` 会被重定向。
- 反代路径：前端不再持有 API Key，所有生图/聊天请求经 `/api/v1/*` 转发到上游。
- 未配置启用模型时，普通生图调用应返回「未配置启用的模型，请联系管理员」。
- 我的画布上云：新建/重命名/删除画布、画布内节点编辑都会自动保存到后端；不同账号在同一浏览器互不可见；换设备登录能看到画布列表与文字结构（图片节点因图床未上云仍可能加载失败）。
- 我的素材上云：新增/编辑/删除走 `/api/assets/me`；不同账号互不可见。
- 生图工作台历史上云：生成完成后顶部历史出现新记录；刷新页面后历史仍在；当前会话能看到缩略图，跨刷新仅显示参数与状态。
- 默认生成尺寸：所有用户首次进入工作台或画布配置节点，默认 size = `auto`、count = `1`。
- 个人中心 `/profile`：顶部卡片显示当前积分、用户名、角色、注册时间；三张统计卡显示累计消耗 / 获赠 / 生图次数；流水表分页正常，颜色区分变动方向。
- 顶栏积分 Tag：普通用户显示蓝色「N 积分」，管理员显示金色「∞」；点击跳转 `/profile`。
- 浏览器本地图片缓存隔离：账号 A 上传的图片，账号 B 登录后看不到（不出现旧 A 的图片缩略图）。
- 管理后台「模型配置」新增配置时：弹窗默认填充图像模型 `gpt-image-2`、文本模型 `gpt-5.4`；填好 Base URL + API Key 后点「获取模型列表」可拉取上游模型，图像/文本模型输入框下拉显示候选；编辑配置时 API Key 留空，也能用库里原 Key 探测；Base URL 或 Key 未填时点击按钮会给出对应提示。
- 长耗时生图：画布或工作台触发 `/api/v1/images/generations`、`/api/v1/images/edits`，即使上游耗时 >30 秒也能成功返回，不再出现 `Internal Server Error / socket hang up`。
- 画布详情页积分 Tag：进入 `/canvas/[id]` 后右上角用户头像左侧出现积分 Tag；普通用户为蓝色「N 积分」、管理员为金色「∞」；点击跳转 `/profile`；生图成功后数值实时减少，生图失败（额度不足/上游报错/未返回图片）不扣减。
