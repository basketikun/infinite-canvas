# 本机部署说明

## 打开方式

双击项目根目录的 `start-local.command`，浏览器会自动打开：

`http://localhost:3000`

保持启动后的终端窗口开启；关闭终端窗口即停止本机服务。

## 已接入模型

- 图片：APIFound `gpt-image-2`，使用自定义 `/v1/imagine2/generate` 调用脚本。
- 视频：火山方舟 `doubao-seedance-2-0-mini-260615`，使用原生异步任务创建与轮询。

API Key、渠道和调用脚本保存在当前浏览器本地，不写入项目文件，也不会提交到 Git。更换浏览器或清理浏览器网站数据后，需要重新配置。

## 视频测试限制

当前验收只使用一次 `480p / 4 秒` 视频任务。后续生成前请在视频创作台确认清晰度和秒数，避免误用更高规格产生额外费用。

## 停止服务

回到启动项目的终端窗口，按 `Control + C`。
